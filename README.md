# Notification Service

A Python-based notification service using FastAPI, Celery, RabbitMQ, Twilio, and more.

## Features
- Email notifications
- SMS notifications
- In-app real-time alerts via Websockets

## Stack
- FastAPI
- Celery + RabbitMQ
- Twilio
- FastAPI-Mail
- Websockets
- PostgreSQL or any DB
